# 为什么要翻译这本书？

本人平时工作中主要用到Go语言，在实际的项目实践过程中，也深知 DDD 的优秀之处，无意中看到本书，打算翻译成为中文，供中文世界的 Gopher 作为。
翻译水平有限，有错误之处，烦请PR修正，也非常欢迎大家一起参与翻译的。
